# Prompt Library

Copy-paste these. Replace <angle brackets>.

## Getting started
```
start my onboarding
```
```
I dropped my bank statement in inbox/ — parse it and update my financial profile.
```
```
give me a full financial health check — cash flow, emergency fund, debt,
investments vs goals. Be brutal about what's off track.
```

## Screening & research
```
Run a fresh stock hunt across <2-3 themes you don't already hold>.
Apply docs/screening-criteria.md strictly. Screen at least 20 names.
For each: PASS or CUT with the specific gate that killed it.
Returning zero passes is a valid answer — do not loosen gates.
```
```
Screen <STOCK> against the quality gate. Pull current fundamentals,
compute 5yr cumulative CFO/PAT explicitly, check promoter pledge trend.
Then check the trap taxonomy — one-quarter spike? inorganic growth?
cyclical peak? Verdict: pass/cut + reason.
```
```
Before I buy <STOCK>: pull the latest data, the community-forum bear case
(ValuePickr for India), and the latest earnings-call guidance.
Is trailing growth durable or an artifact? Is the cash real?
What kills this thesis? Be adversarially skeptical.
```

## Planning & deployment
```
I have ₹<X> to deploy over the next quarter. Given the macro gate and the
verified shortlist, build a staggered deployment plan: entry zones,
stop-losses, position sizes, and the specific trigger for each tranche.
Deploy minimally today. Write it to memory/deployment-plan-<month>.md.
```
```
Write the order card for tranche 1 of <STOCK> per the plan.
```
```
Order filled: <qty> @ ₹<price>. Log it, update the portfolio,
and tell me the stop-loss order to place now.
```

## Monitoring & review
```
Weekly pulse: reprice the portfolio, check every stop and trigger,
flag any thesis-relevant news. Table format.
```
```
<STOCK> reported results. Verify my thesis against the actual numbers:
did the metrics I bought for show up? Verdict: add / hold / kill.
```
```
Monthly review: net worth vs last month, savings rate, portfolio vs plan,
goal progress, what to change. Update all memory files.
```

## Discipline enforcement (use on yourself)
```
I want to buy <STOCK> because <reason>. Steelman the case AGAINST it,
then tell me honestly if I'm chasing.
```
```
I'm down <X>% on <STOCK> and want to average down. Check the original
thesis and stop-loss in memory. Is the thesis intact or am I revenge-buying?
```
```
Challenge me: what's the worst decision in my current portfolio and why?
```
